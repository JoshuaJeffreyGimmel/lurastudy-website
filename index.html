Quick Start

Get Running in One Command

Choose your path — one-line installers or manual Docker setup.

# One-command installer
curl -fsSL https://raw.githubusercontent.com/JoshuaJeffreyGimmel/lurastudy/main/install.sh | sh
# One-command installer (PowerShell)
iwr -useb https://raw.githubusercontent.com/JoshuaJeffreyGimmel/lurastudy/main/install.ps1 | iex
# Manual Docker setup
git clone https://github.com/JoshuaJeffreyGimmel/lurastudy.git
cd lurastudy
cp .env.example .env
docker compose up --build

Then open http://localhost:5173/register — the first user is automatically the admin.

# Cloud AI profile (OpenAI)
docker compose -f docker-compose.yml -f docker-compose.cloud.yml up --build

Set your .env with OpenAI API key, model, and endpoint. See the README for details.

Features

Everything You Need to Study Smarter

26+ features packed into a self-hosted, privacy-first learning platform.

Document Intelligence

Upload PDF, DOCX, PPTX, XLSX, Markdown, HTML, CSV, XML, IPYNB, and plain text. Text is extracted, chunked, and embedded automatically.

Smart Flashcards

AI-generated flashcard decks using structured JSON prompts. SM-2 spaced repetition with 3D flip-card animations and Easy/Got It/Again rating.

AI-Powered Quizzes

Generate multiple-choice, fill-in-the-blank, cloze, and calculation quizzes from your documents. Save attempts and track scoring history.

Chat with Documents

RAG-powered Q&A with your source materials. Ask questions, get answers grounded in your documents. Conversations are saved and resumable.

Chunk Preview

Full transparency — inspect every text chunk extracted from your documents with live search and highlighting. See exactly what the AI sees.

Customizable UI

Customize every UI color with live preview. Choose from preset themes, pick exact hex values, select fonts, and export/import themes as JSON.

Private by Design

Works entirely with Ollama — no data ever leaves your machine. Per-user data isolation with JWT authentication and encrypted passwords.

One-Command Deploy

Single Docker Compose command spins up PostgreSQL + Backend + Frontend. One-liner installers for Linux, macOS, and Windows.

How It Works

From Upload to Mastery in Minutes

Three simple steps to transform your study materials into an interactive learning experience.

01

Upload Your Materials

Drop in PDFs, lecture slides, textbooks, notes — any format. LuraStudy extracts text, splits it into chunks, and generates vector embeddings automatically.

02

AI Processes Everything

Your chosen LLM analyzes the content. Documents are indexed in a vector database for semantic search. Knowledge bases are built from your material.

03

Study Smarter

Generate flashcards with SM-2 spaced repetition, take AI-powered quizzes, or chat naturally with your documents. Track your progress and master the material.
